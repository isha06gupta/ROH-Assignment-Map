What is CMM ROH/POH Wagon Tracking?
🚂 Wagon in Service

A freight wagon runs kilometres on Indian Railways network. Every wagon has a schedule for Periodic Overhaul (POH) and Return to Home (ROH) based on distance covered, age, and condition.

↓
📋 Nominated for Maintenance

CMM nominates wagons for ROH or POH. The wagon gets a Rake ID and is tracked from its current station through to its destination depot.

↓ Splits into two tracks
🔧 ROH — Return Overhaul

Scheduled every ~18 months. Lighter maintenance. Wagon goes to its designated ROH depot. Overdue Days column flags how late it is.

🏭 POH — Periodic Overhaul

Scheduled every 4.5 years. Complete overhaul. Wagon goes to POH depot. Distance and Expected Arrival track transit progress.

↓
📍 In Transit / At Depot

Each rake record shows Current Station → Next Station → Destination. Distance (km) and Expected Travel Time (hrs) tell you how far out the rake is from its ROH depot.

↓
✅ Depot Perspective (Heat Map)

Which depots have the most rakes incoming? The heat map shows each ROH depot as a circle — bigger/redder = more rakes pending at that location.

⚠️ Overdue Perspective

Rakes where Overdue Days > 0 are late for maintenance. The higher the number, the more critical. Shown in red on the Booked To depot tab.

NKJRH_ROH CSV — Column Dictionary (11 columns)
ID
ROH Depot
The depot code where the rake is assigned for ROH maintenance (e.g. NKJRH, UDLRH, TKDRH).
★ KEY — used to group rakes per depot and plot heat map circles.
ID
Rake ID
Unique identifier for the rake/consist being sent for maintenance.
Primary key for each record; shown in popup on map.
OPS
Number of Wagons
How many individual wagons are in this rake.
Workload metric — a rake of 30 wagons is heavier than one of 3.
STATUS
Overdue Days
How many days past the scheduled ROH date this rake is. 0 = on time, positive = overdue.
★ Critical — drives the overdue KPI and red highlighting on the map.
GEO
Current Station
Station code where the rake is right now.
Live location tracking; map origin point for wagon dots.
GEO
Next Station
The next station the rake is routed through en route to destination.
Routing visibility; intermediate waypoint.
GEO
Destination
Final destination depot code (FD suffix format, e.g. ADTPFD).
Confirms which depot the rake is booked to; shown in wagon popup.
DATE
Last Updated
Timestamp of when this record was last refreshed in the system.
Data freshness check; filter for stale records.
OPS
Distance (km)
Kilometres remaining from current station to destination depot.
Transit progress; 0 = rake already at depot.
OPS
Expected Travel Time (hrs)
Estimated hours to reach destination from current station.
Arrival planning; depot capacity scheduling.
DATE
Expected Arrival
Calculated datetime when rake is expected to arrive at the depot.
★ Depot scheduling — sort by this to see what's arriving soonest.
